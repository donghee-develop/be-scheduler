# 일정 관리 프로젝트

## 1. 프로젝트 소개

스프링 부트 기반의 일정 관리 웹 페이지

## 2. 개발 기간 및 개발자


개발 기간 : 25.03.19 ~ 25.03.25

개발자 : 김동희

## 3. 개발 환경 및 기술

사용 기술 : Java SpringBoot, Git, GitHub, Vue3, Postman (API Doc)

자바 버전 : Java 17

프로젝트 : SpringBoot

빌드 도구 : Gradle 

DB : mysql 8.0.36, JDBC

의존성 : Lombok, Valid 외 DB 관련 의존성 추가

ERD 설계 : https://www.erdcloud.com/d/Q3ZTnSSAXxtr52o4Y
